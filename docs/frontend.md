# Frontend

Next.js 15 App Router - component guide, hooks, and UX decisions will be documented here as the frontend is built.

## Folder Structure

`app/` - Next.js App Router pages (home, login)

`components/` - all UI components; each in its own folder with its CSS Module alongside it

`hooks/` - custom hooks (watch history, content fetching)

`lib/` - typed fetch wrappers for the backend API

`types/` - shared TypeScript interfaces

`styles/` - global CSS reset and base styles
